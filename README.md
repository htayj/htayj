## Things you can actually use right now

- [`manna-cadet`](https://github.com/htayj/manna-cadet) — Space Cadet-inspired keyboard configuration that can simulate all four Space Cadet bucky bits (Control, Meta, Super, and Hyper), plus an extra “Alt” modifier that normally acts as Meta but can be bound separately, for example by a window manager, without interfering with the other bucky bits. It also supports Shift, Greek, and Top.
- [`DEC-Fonts`](https://github.com/htayj/DEC-Fonts) — recreations of historical VT220 font ROM dumps, usable in BDF, OTB, and PSF formats on modern computers.

## Things you can actually read right now

## Ongoing development

- [`clawmacs`](https://github.com/htayj/clawmacs) — a Common Lisp agent harness and agentic coding tool using McCLIM. It attempts to synthesize the strengths of Emacs and Lisp Machines into agentic editors.
- [`flaghack`](https://github.com/htayj/flaghack) — a roguelike similar to NetHack.
- [`lisp-machine-container-museum`](https://github.com/htayj/lisp-machine-container-museum) — containers for exploring historical Lisp machines. It aims to make Lisp Machine emulators more accessible.
- [`General-Corpus`](https://github.com/Kyvero-Press/General-Corpus) — a collection of public-domain Middle English texts converted from various XML-based formats to TeX for print-on-demand or publication. It aims to provide an alternative to the low-quality public-domain republications of obscure materials available on Amazon.
