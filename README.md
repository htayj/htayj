## Things you can actually use right now
- [`manna-cadet`](https://github.com/htayj/manna-cadet) — Space Cadet-inspired keyboard configuration - allows simulating all 4 (C-M-s-H) space cadet bucky bits (plus one extra, "Alt" which is normally treated as Meta but can be bound separately, to a window manager for example, without interferring with the other bucky bits), plus plus shift, greek, and top. 
- [`dec-fonts`](https://github.com/htayj/DEC-Fonts) - recreation of historical VT220 fonts ROM dumps usable in bdf, otb, and psf formats for modern computers

## Things you can actually read right now

## ongoing development

- [`clawmacs`](https://github.com/htayj/clawmacs) — common lisp agent harness & agentic coding tool using mcclim. Attempting to synethesize the strengths of Emacs and Lisp Machines into agentic editors
- [`flaghack`](https://github.com/htayj/flaghack) — a roguelike similar to nethack
- [`lisp-machine-container-museum`](https://github.com/htayj/lisp-machine-container-museum) — containers for exploring historical Lisp machines. Attempting to make running lispm emulators more accessible
- [`General-Corpus`](https://github.com/Kyvero-Press/General-Corpus) - a collection of public domain Middle English texts converted from various XML-based formats to TeX for print-on-demand or publication - attempting to provide an alternative to low quality public domain republications available on amazon for obscure materials.
